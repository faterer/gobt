## 🔍 代码亮点

### 创建Torrent的关键代码

```go
// 1. 构建Go数据结构
torrentDict := map[string]interface{}{
    "announce": "http://tracker.example.com:6969/announce",
    "info": map[string]interface{}{
        "name": "ubuntu-22.04-desktop-amd64.iso",
        "length": int64(3520000000),
        "piece length": int64(262144),
        "pieces": pieceHashes, // 二进制SHA1哈希
    },
    "creation date": time.Now().Unix(),
    "created by": "gop2p/4.2.0",
}

// 2. 使用Bencode编码器编码
encoder := bencode.NewEncoder()
encoded, _ := encoder.Encode(torrentDict)

// 3. 保存为文件
ioutil.WriteFile("ubuntu-22.04.torrent", encoded, 0644)
```

### 解析Torrent的关键代码

```go
// 1. 读取文件
data, _ := ioutil.ReadFile("ubuntu-22.04.torrent")

// 2. 使用Bencode解码器解析
decoder := bencode.NewDecoder(bytes.NewReader(data))
decoded, _ := decoder.Decode()

// 3. 类型转换为字典
torrentDict := decoded.(map[string]interface{})

// 4. 计算Info Hash (这在BitTorrent中必需)
infoDict := torrentDict["info"].(map[string]interface{})
infoEncoded, _ := encoder.Encode(infoDict)
infoHash := sha1.Sum(infoEncoded)  // 产生20字节的哈希

// 5. 提取信息
announce := torrentDict["announce"].(string)
name := infoDict["name"].(string)
pieceLength := infoDict["piece length"].(int64)
```

## 💡 关键学习点

### 1. Bencode格式优势

✅ **紧凑** - 比JSON/XML小30-50%  
✅ **简单** - 四种基本类型就够了  
✅ **无歧义** - 严格定义，易于实现  
✅ **递归支持** - 天然支持嵌套结构  
✅ **二进制安全** - 可存储任意字节

### 2. Info Hash的重要性

```
Info Hash = SHA1(Bencode(info部分))

用途:
• 唯一识别一个torrent
• 在Tracker中注册
• 在DHT中查询Peer
• BitTorrent协议中的必需字段
```

### 3. 单文件 vs 多文件模式

```
单文件模式:
info {
  "name": "file.iso",
  "length": 1000000,        ← 直接在info中
  "piece length": 262144,
  "pieces": [binary]
}

多文件模式:
info {
  "name": "folder",
  "files": [               ← 嵌套列表
    {
      "length": 500000,
      "path": ["subdir", "file1.iso"]
    },
    {
      "length": 500000,
      "path": ["subdir", "file2.iso"]
    }
  ],
  "piece length": 262144,
  "pieces": [binary]
}
```
